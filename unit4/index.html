Unit 4: Web Graphics

Overview

There are four modules in this unit. In the first module you will be introduced to using graphics in web design and learn about what constitutes effective use of web graphics. In the second module, you will acquire digital images and then learn how to crop and resize the images so they can be displayed on web pages. In the third module, you will utilize simple drawing tools and layers in graphics software to create simple web graphics and buttons. In the fourth module, you will learn about selection tools, tools for image manipulation, and blending techniques. The final project will be to create a banner image to be used on your portfolio web page.
