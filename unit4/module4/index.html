UNIT 4

Module 4: Creating a Web Page Banner

Overview

In this module, you will learn how to utilize selection tools to isolate parts of an image. You will then learn how to apply layer effects and blend objects on different layers to create digital collages. Using these skills, you will apply design concepts from Unit 1 to create a web banner for your portfolio site.
