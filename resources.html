Student Resources

This page includes dozens of tutorials, reference guides, and tools to help beginners as well as advanced coders to hone their craft. Best of all, everything listed here is free! Explore these sites to discover new ways to expand your web design and development skills.

Learning about HTML

  • W3Schools HTML Tutorial
  • W3Schools HTML Reference: tags ordered alphabetically or tags ordered by function
  • HTML 5 Spec - the definitive source, but not easy reading!

Learning About CSS

  • W3Schools CSS Tutorial
  • W3Schools CSS Properties Reference
  • CSS Zen Garden - a site that demonstrates the potential of CSS

Learning About Javascript

  • W3Schools Javascript Tutorial
  • W3Schools Javascript Reference

Learning About Web Accessibility

  • WebAIM Introduction to Web Accessibility
  • Developing Accessible Websites - the University of Washington's official site on technology accessibility includes a variety of accessibility-related tips and techniques for web designers and developers

Learning About Everything

  • W3Schools - in addition to their HTML, CSS, and JavaScript tutorials, they also have excellent tutorials on dozens of advanced topics in web design and development—great for any student who's looking to expand their skills
  • Code Avengers - free tutorials on HTML, CSS, & JavaScript
  • Code Academy - free tutorials for learning web design plus several popular programming languages

Learning About Graphics

  • 50 Totally Free Lessons in Graphic Design Theory

Learning About Copyright

  • U.S. Copyright Office - Frequently Asked Questions
  • Cyberbee Interactive Copyright Questions and Answers

Color Tools

  • HTML Color Picker
  • Wikipedia: Web Colors
  • Colour Contrast Analyser - free downloadable software for Windows and Mac, can be used to check contrast between foreground and background colors anywhere on the computer screen

Validators and Accessibility Checkers
