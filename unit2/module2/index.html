UNIT 2

Module 2: Basic HTML Markup

Overview

In this module, you will begin to learn about the basics of HTML coding. You will receive an introduction to HTML tags, including opening and closing tags, elements, and attributes. Next, you will learn the essential tags for a website and experience the difference between the head and body sections of a web page. You will then create a "bare bones" template and use this to create an index page and a few sub-pages for your portfolio. You will then be introduced to some of the most common HTML elements and begin adding content to your portfolio index page.
