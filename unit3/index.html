Unit 3: Formatting Web Pages with Style Sheets

Overview

This unit is the sequel to the proceeding unit, in which you learned the basics of HTML code, but focused solely on the content and structure of web documents. In this unit, you learn how to control the visual presentation of web documents using Cascading Style Sheets (CSS). This unit can be done entirely with a web browser and simple text editor.
