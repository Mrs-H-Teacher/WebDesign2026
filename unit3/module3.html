UNIT 3

Module 3: Typography in CSS

Overview

In this module, the two lesson help students to first understand typography on web pages, then apply that knowledge to their portfolio sites.

Lesson 1: Understanding Typography in CSS

  • Estimated time required: 30 - 60 minutes
  • Link to student Lesson 1 page

Tips for Delivering This Lesson

  • This lesson includes a short assessment based on assigned reading. As in similar lessons earlier in the curriculum, this assessment could be conducted over email, or using an online survey or assessment tool.
  • Travis Hitchcock has produced an interesting Pretzl Presentation on Typography that you may want to share and discuss with your students at the start of this lesson.

Lesson 2: Applying Typography in CSS

  • Estimated time required: 60 - 90 minutes
  • Link to student Lesson 2 page

Tips for Delivering This Lesson

  • This is a very long lesson and may require multiple class sessions. The most crucial activities are those numbered 8 through 13, in which students add a pair of font-family properties to their external CSS file. If your studentsâ€™ interest is waning in this lesson, consider how you can trim down the earlier activities and focus on these latter activities.

Example Output

When this lesson is complete, students will have applied typography-related styles to their site by changing the external CSS file. These changes should be visible on any web page within their site, including index.html.
